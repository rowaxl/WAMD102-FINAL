Insertion Sort #
Overview
  • 1Set a marker for the sorted section after the first number in the list.
  • 2Select the first unsorted number.
  • 3Swap this number to the left until it arrives at the correct sorted position
  • 4Advance the marker to the right one position
  • 5Repeat steps 2 through 4 until the unsorted section is empty
Specification
Best case in time Worst case in time Average case in time Space complexity
\[O(n)\] \[O(n^2)\] \[O(n^2)\] \[O(1)\]

Selection Sort #
Overview
  • 1Set a marker for the unsorted section at the front of the list.
  • 2Compare all unsorted numbers in order to select the smallest one.
  • 3Advance the marker to the right one position.
  • 4Swap this number with the first number in the unsorted section.
  • 5Repeat steps 2 - 4 until one number remains in the unsorted section.
Specification
Best case in time Worst case in time Average case in time Space complexity
\[O(n^2)\] \[O(n^2)\] \[O(n^2)\] \[O(1)\]

Bubble Sort #
Overview
  • 1Starting with the first element(index = 0), compare the current element with the next element of the array.
  • 2If the current element is greater than the next element of the array, swap them.
  • 3If the current element is less than the next element, move to the next element. Repeat Step 1.
Specification
Best case in time Worst case in time Average case in time Space complexity
\[O(n)\] \[O(n^2)\] \[O(n^2)\] \[O(1)\]

Merge Sort #
Overview
  • 1Break array into the two parts until it has only a single element.
  • 2Merge contiguous sub-arrays with sorting elements.
  • 3Continue step 2 while all subarrays merged.
Specification
Best case in time Worst case in time Average case in time Space complexity
\[O(n \log n)\] \[O(n \log n)\] \[O(n \log n)\] \[O(1)\]

Quick Sort #
Overview
  • 1Find a pivot item in the array.
  • 2Start a pointer (the left pointer) at the first item in the array.
  • 3Start a pointer (the right pointer) at the last item in the array.
  • 4While the left value is less than the pivot value, move the left pointer to the right.
  • 5While the right value is greater than the pivot value, move the right pointer to the left
  • 6If the left pointer <= the right pointer, then swap the values at these locations in the array.
  • 7Move the left pointer to the right by one and the right pointer to the left by one.
  • 8If the left pointer and right pointer don’t meet, go to step 1.
Specification
Best case in time Worst case in time Average case in time Space complexity
\[O(n^2)\] \[O(n \log n)\] \[O(n \log n)\] \[O(\log n)\]
