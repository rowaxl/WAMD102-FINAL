Bubble Sort
Overview
  • 1Starting with the first element(index = 0), compare the current element with the next element of the array.
  • 2If the current element is greater than the next element of the array, swap them.
  • 3If the current element is less than the next element, move to the next element. Repeat Step 1.

Merge Sort
Overview
  • 1Break array into the two parts until it has only a single element.
  • 2Merge contiguous sub-arrays with sorting elements.
  • 3Continue step 2 while all subarrays merged.
